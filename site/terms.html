Service Description

Pulse is a free SMS-based service that provides curated event and nightlife recommendations for New York City neighborhoods. You text a neighborhood name, and Pulse responds with event picks sourced from public listings across the web.

How to Use Pulse

  • Text any NYC neighborhood name to (833) 785-7300 to receive event recommendations.
  • Reply with a number to get details on a specific event.
  • Reply "more" for additional picks, or filter by category (e.g., "comedy," "free," "late night").
  • Reply STOP at any time to stop receiving messages.
  • Reply HELP for support information.
