From SMS to Response in 9 Steps

$0 <1ms

03 / Agent Brain

Two Tools, One Chat Session

The agent brain is a Gemini chat session with structured tool calling. No regex. No intent classifiers. Just two tools.

3-File Split
agent-brain.js
orchestrator ~450 lines
brain-llm.js
LLM calls + compose ~550 lines
brain-execute.js
tool execution + pools ~530 lines
No circular deps.
Re-exports preserve all existing imports.
2 Tools
search_events
  neighborhood string
  category enum
comedy, jazz, live_music, dj, trivia, film, theater, art, dance, community, food_drink, spoken_word, classical, nightlife
  categories array
  free_only boolean
  time_after string HH:MM
  date_range enum
today, tomorrow, this_weekend, this_week, next_week
  intent enum
new_search, refine, pivot, more, details
  pick_reference string
respond
  message string max 480 chars
  intent enum
greeting, thanks, farewell, off_topic, clarify, acknowledge
P1: Tool call params ARE the state. Filters, intent, and neighborhood come from structured tool calls — never parsed from free text.
