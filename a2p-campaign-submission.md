## Campaign Information

### Description

Pulse is a free SMS-based service that recommends NYC nightlife and events. Users discover the service through our website at https://jkoufopoulos.github.io/pulse-sms/, which displays the phone number (646) 722-6926 alongside full consent disclosures including message frequency, data rates, opt-out instructions, and links to our privacy policy and terms. Users opt in by texting START or any NYC neighborhood name to the number. Pulse replies with 2-3 curated event picks. Users control the conversation — we only reply to their messages and never text first. Message frequency varies based on user requests. Reply STOP to opt out. Reply HELP for help.

## End User Consent

### How do end-users consent to receive messages?

Users find Pulse through our public website at https://jkoufopoulos.github.io/pulse-sms/. The website prominently displays the phone number (646) 722-6926 with the following consent disclosures visible before the user texts: brand identification (Pulse), a description of the service (SMS event recommendations), message frequency disclosure (varies based on user requests), "Msg & data rates may apply", "Consent is not a condition of purchase", opt-out instructions (reply STOP), help instructions (reply HELP), and links to our Privacy Policy and Terms of Service. Users initiate contact by texting START or any NYC neighborhood name to (646) 722-6926. This is a user-initiated, conversational service — Pulse only replies to inbound messages and never sends unsolicited texts. The opt-in keyword is START. Users can opt out at any time by replying STOP.
