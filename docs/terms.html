Program Name

Pulse SMS — NYC Event Discovery

Program Description

Pulse is an SMS-based service that recommends events, nightlife, and things to do in New York City. Text a neighborhood name to receive curated event picks for tonight.

Message Frequency

Message frequency varies based on user interaction. You will only receive messages in direct response to texts you send. Pulse does not send unsolicited promotional messages.

Message and Data Rates

Message and data rates may apply. Contact your wireless carrier for details about your text messaging plan.

How to Get Help

Text HELP to our number at any time for instructions and support information.

How to Opt Out

Text STOP to our number at any time to stop receiving messages. You will receive a confirmation and no further messages will be sent. You can text START to re-subscribe at any time.

Supported Carriers

Pulse SMS is available on all major US wireless carriers including AT&T, T-Mobile, Verizon, and others.

Disclaimer

Event information is sourced from third-party websites and may not always be accurate or up-to-date. Pulse provides recommendations on a best-effort basis and is not responsible for event cancellations, incorrect details, or changes. Always verify event details before attending.
